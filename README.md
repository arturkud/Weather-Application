# Weather-Application
Weather Application with API integration built with HTML, CSS, and JavaScript

## Description
This application retrieves and displays weather data for user-entered locations.

## How to you the application
- Sign up at https://openweathermap.org/ and generate a free API key
- Open the `script.js` file
- Replace the value of the `key` variable with your API key

## How to run the application
- Open `index.html` file to run the application
- Enter the location in the input field and click `enter`

#Technologies used
- HTML, CSS, JavaScript
